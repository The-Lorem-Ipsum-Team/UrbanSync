══════════════════════════════════════════════════════════════
PROJECT OVERVIEW
══════════════════════════════════════════════════════════════

UrbanSync fuses two municipal datasets to calculate a Civic Friction Score (CFS) for every open infrastructure complaint in Khon Kaen city, Thailand. The CFS re-ranks the complaint dispatch queue from First-In-First-Out to Mobility-ROI order — fixing complaints that block the most traffic first.

CFS FORMULA:
  CFS = complaint_severity_score × traffic_multiplier

  complaint_severity_score: integer 1–10
  traffic_multiplier: float
    >130,000 vehicles/day  → 3.0
    80,000–130,000         → 2.0
    30,000–80,000          → 1.5
    <30,000                → 1.0

MODULE 1 — src/cv_pipeline.py
══════════════════════════════════════════════════════════════

PURPOSE: Process CCTV videos to count unique vehicles crossing tripwire lines.

IMPORTS NEEDED: ultralytics, supervision, cv2, pathlib, json, pandas, numpy, tqdm

FUNCTION 1: load_tripwire_config(config_path, video_id, frame_width, frame_height) -> dict
  Load config/video_tripwires.json. Look up video_id key; fall back to "default".
  Convert fractional coordinates to pixel coordinates:
    pixel_x = fraction_x * frame_width
    pixel_y = fraction_y * frame_height
  Return dict with keys: line_A_points, line_B_points, direction_A_label, direction_B_label
  where *_points are lists of (int, int) tuples.

FUNCTION 2: process_video(video_path, model_path, tripwire_config_path,
                           video_id=None, location_name="unknown") -> dict
  Steps:
  a. Load YOLO model: model = YOLO(model_path)
  b. Open video with cv2.VideoCapture to get frame_width, frame_height, fps, total_frames
  c. Load tripwire config using load_tripwire_config()
  d. Define two supervision.LineZone objects from the pixel coordinates
     LineZone(start=sv.Point(x1,y1), end=sv.Point(x2,y2))
  e. Track with: results = model.track(source=str(video_path), tracker="bytetrack.yaml",
                                        persist=True, stream=True, verbose=False)
  f. For each result frame:
     - Convert to supervision: detections = sv.Detections.from_ultralytics(result)
     - Filter classes: keep only class_ids in [2,3,5,7]
       (COCO: 2=car, 3=motorcycle, 5=bus, 7=truck)
     - Map classes: {2:'Car', 3:'Motorcycle', 5:'Truck', 7:'Truck'}
     - Only keep detections with tracker_id not None
     - Update both LineZone objects: zone_A.trigger(detections), zone_B.trigger(detections)
     - Track unique crossings per class using sets:
         crossed_A: dict mapping track_id -> vehicle_class
         crossed_B: dict mapping track_id -> vehicle_class
       Add to crossed_A only when zone_A.in_count increases for that track_id
  g. Count final unique vehicles:
     For each crossed dict, count by class.
  h. Return dict:
     {
       "video_id": str,
       "location_name": str,
       "direction_A_label": str,
       "direction_B_label": str,
       "direction_A_total": int,
       "direction_B_total": int,
       "bidirectional_total": int,
       "car_count": int,
       "motorcycle_count": int,
       "truck_count": int,
       "total_unique_vehicles": int,
       "processed_frames": int,
       "video_duration_seconds": float,
       "source_file": str
     }

FUNCTION 3: process_all_videos(video_dir, model_path, tripwire_config_path,
                                video_location_map=None) -> pd.DataFrame
  - video_location_map: optional dict mapping filename stem -> location_name string
  - Find all .mp4, .avi, .mkv, .mov files in video_dir
  - Process each with process_video(), catching exceptions per video
  - Show tqdm progress bar
  - Collect results into a list of dicts
  - Save to outputs/video_counts.csv
  - Print summary table: location | bidirectional | car | motorcycle | truck
  - Return DataFrame

FUNCTION 4: download_youtube_videos(playlist_url, output_dir) -> list[Path]

MODULE 4 — src/spatial_join.py
══════════════════════════════════════════════════════════════

PURPOSE: Join each complaint to its nearest traffic GPS checkpoint using
         district bounding box centroids and KDTree nearest-neighbor search.

IMPORTS NEEDED: pandas, numpy, pathlib, json, math
                scipy.spatial.KDTree

FUNCTION 1: haversine_km(lat1, lng1, lat2, lng2) -> float
  - Standard haversine formula, returns kilometers
  - R = 6371.0
  - All inputs converted to radians internally

FUNCTION 2: load_district_bounds(bounds_path) -> dict
  - Load config/district_bounds.json
  - Return dict as-is

FUNCTION 3: district_to_centroid(district_str, district_bounds) -> tuple[float, float]
  - Compute centroid of bounding box for given district key
  - centroid_lat = (lat_min + lat_max) / 2
  - centroid_lng = (lng_min + lng_max) / 2
  - If district_str not in bounds: use "ไม่ระบุ" entry (KK city center)
  - Return (lat, lng)

FUNCTION 4: build_complaint_coordinates(complaints_df, district_bounds) -> np.ndarray
  - For each row in complaints_df, get centroid from district_to_centroid()
  - Return numpy array of shape (N, 2) with columns [lat, lng]
  - This vectorizes the centroid lookup before KDTree query

FUNCTION 5: join_complaints_to_traffic(complaints_df, traffic_df,
                                        district_bounds_path) -> pd.DataFrame
  - Load district bounds
  - Build complaint coordinate array using build_complaint_coordinates()
  - Build KDTree from traffic checkpoint coordinates:
      checkpoint_coords = traffic_df[['Lat', 'Lng']].values
      tree = KDTree(checkpoint_coords)
  - Query tree: distances, indices = tree.query(complaint_coords, k=1)
    distances here are Euclidean in lat/lng degrees — convert to km:
      km_distances = distances * 111.0  (1 degree ≈ 111 km approximation)
  - Map each complaint to its nearest checkpoint:
      nearest checkpoint columns to add to complaints_df:
        nearest_checkpoint_id    (str, e.g. "CP_01")
        checkpoint_road_name     (str, เส้นทาง)
        checkpoint_location      (str, ตำแหน่งติดตั้งเครื่องวัด)
        checkpoint_distance_km   (float, 1 decimal)
        traffic_multiplier       (float)
        daily_vehicle_count      (int, รวมต่อวัน)
        weighted_volume          (float)
        checkpoint_lat           (float)
        checkpoint_lng           (float)
  - Print: "Spatial join complete. N complaints matched."
  - Print: "Mean distance to nearest checkpoint: X.X km"
  - Print per-district complaint counts
  - Return joined DataFrame

FUNCTION 6: run_spatial_join(complaints_path, traffic_path, district_bounds_path) -> pd.DataFrame
  - Load both CSVs
  - Run join_complaints_to_traffic()
  - Save outputs/complaints_with_traffic.csv
  - Return joined DataFrame

PURPOSE: Compute CFS for all open complaints, rank them, produce output files.

IMPORTS NEEDED: pandas, numpy, pathlib, json, datetime

FUNCTION 1: compute_cfs(df) -> pd.DataFrame
  - Filter: open_df = df[df['is_open'] == True].copy()
  - cfs_score = severity_score * traffic_multiplier (round to 2 dp)
  - cfs_tier:
      cfs_score >= 24 → "critical"
      cfs_score >= 15 → "high"
      cfs_score >= 8  → "medium"
      else            → "low"
  - priority_rank = cfs_score rank descending, method='first' (no ties)
  - Sort by priority_rank ascending
  - Print tier distribution table
  - Return open_df with new columns

FUNCTION 2: compute_fifo_vs_cfs_comparison(df) -> pd.DataFrame
  - df must have วันที่รับเรื่อง and priority_rank columns
  - fifo_rank = rank by วันที่รับเรื่อง ascending (oldest=rank 1)
  - rank_change = fifo_rank - priority_rank
    (positive = CFS promotes this complaint; negative = CFS deprioritizes)
  - rank_change_label:
      rank_change > 100  → "major upgrade"
      rank_change > 20   → "upgrade"
      rank_change < -20  → "downgrade"
      else               → "similar"
  - Save outputs/fifo_vs_cfs_comparison.csv
  - Print top 10 biggest rank upgrades with their details
  - Print: "Biggest beneficiaries of CFS vs FIFO:"
  - Return df with rank comparison columns

FUNCTION 3: format_queue_record(row) -> dict
  - Convert a single DataFrame row to the output record format:
    {
      "rank": int(row.priority_rank),
      "complaint_id": str(row.เลขคำร้อง),
      "complaint_type": str(row.ประเภทคำร้อง),
      "description": str(row.เรื่องร้องทุกข์),
      "district": str(row.เขต),
      "community": str(row.ชุมชน),
      "days_open": int(row.days_open) if not pd.isna(row.days_open) else 0,
      "severity_score": int(row.severity_score),
      "traffic_multiplier": float(row.traffic_multiplier),
      "cfs_score": float(row.cfs_score),
      "cfs_tier": str(row.cfs_tier),
      "nearest_road": str(row.checkpoint_road_name),
      "daily_vehicles": int(row.daily_vehicle_count),
      "date_received": str(row.วันที่รับเรื่อง)
    }

FUNCTION 4: generate_queue_output(ranked_df) -> dict
  - Apply format_queue_record to each row
  - Result dict:
    {
      "generated_at": datetime.datetime.now().isoformat(),
      "total_open": len(ranked_df),
      "queue": [list of formatted records]
    }
  - Save outputs/ranked_queue.json (ensure_ascii=False, indent=2)
  - Save outputs/ranked_queue.csv
  - Print: "Queue saved. Top complaint: {type} | CFS: {score} | District: {district}"
  - Return result dict

FUNCTION 5: compute_statistics(df, full_df) -> dict
  - df = open complaints with CFS; full_df = all complaints
  - stats = {
      "total_open": len(df),
      "total_closed": int((full_df['is_open'] == False).sum()),
      "critical_count": int((df['cfs_tier'] == 'critical').sum()),
      "high_count": int((df['cfs_tier'] == 'high').sum()),
      "medium_count": int((df['cfs_tier'] == 'medium').sum()),
      "low_count": int((df['cfs_tier'] == 'low').sum()),
      "mean_days_open": round(float(df['days_open'].dropna().mean()), 1),

    }
  - Save outputs/statistics.json
  - Return stats
